# ADR-0011：代码注释统一使用中文

> 状态：当前有效
> 适用范围：生产代码、测试和接口描述
> 最后校准：2026-07-10
> 对应代码：`MyBlog-springboot-v2/src/`、`frontend/apps/`
> 权威程度：ADR

## 背景

项目文档和业务术语以中文为主。中英文混杂且仅复述代码的注释会降低可读性。

## 决策

Javadoc、业务注释、测试场景说明和 OpenAPI 描述统一使用简洁中文。技术名词、代码标识、协议字段、配置键和错误码保留原始英文。

注释解释业务约束、边界条件或设计原因，不重复方法名和语句表面含义。

## 结果

代码语义与项目文档保持一致。具体写法见 `../rules/comment-style.md`。
