# ADR-0011: 注释统一使用中文

- 状态：accepted
- 日期：2026-04
- 决策者：项目负责人

## 背景

V1 注释中英文混杂，部分英文注释只是翻译代码本身，无业务价值。维护者（本人）母语为中文，AI 助手处理中文上下文也无障碍。

## 决定

V2 所有 Javadoc、字段注释、行内注释、测试场景注释、OpenAPI 描述统一中文。

允许保留英文的部分：
- 技术名词（Spring、JWT、MyBatis-Plus 等）
- 数据库字段名、配置 key、HTTP header、错误码等固定标识
- 包/类/方法名

## 理由

- 项目主语言中文，业务术语本就中文
- 注释表达业务语义，中文表达更精准
- 与英文混杂反而损害可读性

## 后果

正面：业务语义清晰，维护成本低
负面：若未来引入外部协作者（非中文母语）需提前沟通
