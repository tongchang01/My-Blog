## Overview

MyBlog is a full-stack blog system with:

- a public blog frontend
- an admin dashboard
- a Spring Boot backend

Project structure:

```text
My-Blog
├─ MyBlog-vue
│  ├─ MyBlog-blog
│  └─ MyBlog-admin
└─ MyBlog-springboot
```

For full documentation, choose your preferred language above.
